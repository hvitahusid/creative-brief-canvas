Creative Brief
==============

### Commands:

    # Run test server (on port 3000).
    # This will also take ware of installing any meteorite packages.
    mrt
    
    # Deploy to server.
    # Bundle/zip - stop server - upload zip to server, unzip and prepare - start server again.
    fab bundle stop deploy start
